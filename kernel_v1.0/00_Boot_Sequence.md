# 00 — Boot Sequence

Welcome to Kernel v1.0 — the Recovery Kernel "boot sequence" introduces the project's intent, scope, and dedication.

Purpose
- Provide a community-oriented, open reference for understanding frameworks and correlations that different communities use around recovery, resilience, and personal genomics.

Dedication
- Dedicated to anyone working towards safer, evidence-based, and community-centered recovery approaches.

How to read this book
- Each chapter is modular. Read sequentially for a guided experience or jump to the sections most relevant to you.
- Chapters include technical context, ethical disclaimers, and case-study summaries (fictionalized where necessary).
