Rewritten in English
A lot of people have asked whether this was generated by AI. The real answer is more nuanced: the code, the templates, the structure — all of that was designed and co‑authored by me, using AI as a tool. After that, the AI drafted the book, and I went back through it manually, editing, refining, and using prompt engineering to guide the details.

Everything — including the templates, the framework, and even the “write your own chapter” system — is included in the GitHub package if you explore it deeply enough. There’s even a prototype workflow where you can submit DNA data and a medication’s package insert; the AI checks for compatibility and offers dosing guidance so you’re not given a dose that’s too strong or a medication that could affect you negatively.

The Protocols section is simply “notes to live by when you’re dealing with addiction.” The rest of the GitHub, if you dig into it, contains much more — including the scientific whitepapers the project is built on.

Versión en Español
Muchos han preguntado si esto fue generado por IA. La respuesta real es más matizada: el código, las plantillas y la estructura fueron diseñados y coescritos por mí, usando la IA como herramienta. Después de eso, la IA redactó el libro, y luego yo lo edité manualmente, afiné el contenido y utilicé ingeniería de prompts para guiar los detalles.
Todo —incluyendo las plantillas, el marco completo y hasta el sistema para “escribir tu propio capítulo”— está incluido en el paquete de GitHub si lo exploras lo suficiente. Incluso hay un flujo de trabajo prototipo donde puedes enviar datos de ADN y el prospecto de un medicamento; la IA revisa la compatibilidad y ofrece orientación sobre la dosis para evitar que recibas una concentración demasiado alta o un fármaco que pueda afectarte negativamente.
La sección de Protocolos es simplemente “notas para vivir cuando lidias con la adicción”. El resto del GitHub, si lo exploras, contiene mucho más —incluidos los artículos científicos en los que se fundamenta el proyecto.
